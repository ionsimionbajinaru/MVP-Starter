# MVP Starter

A lightweight, framework-free starter template for founders who need to validate ideas quickly. It includes a landing page, a multi-step signup form, and a simple dashboard UI so you can start testing demand right away.

## What this project is

MVP Starter is a static frontend template built with:

- Vanilla JavaScript
- HTML
- CSS

It is intentionally simple, modular, and easy to extend.

## How to run it

Because this is a static project, you can run it with any local static server.

### Quick preview from repository root

If your preview environment serves the repository root, open `/index.html` and it will redirect to `mvp-starter/index.html` automatically.

### Option 1: Python

```bash
cd mvp-starter
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000/index.html`
- `http://localhost:8000/dashboard.html`

### Option 2: VS Code Live Server

Open the `mvp-starter` folder and start Live Server from `index.html`.

## Use case for founders building MVPs

This template is ideal when you want to:

- Launch a clean landing page fast
- Capture early interest with a structured signup flow
- Present simple traction data in a dashboard-style UI
- Iterate without the overhead of frameworks
